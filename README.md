# Service Business Website w/Quote and Price Calculator

A sample service based business website that includes a quote/price calculator built in so customers who need to just know what the service cost without waiting for a response or having to call the business itself.

## Sample Business
- Name: Evergreen Property Services
- Services: [
    - Lawn Mowing
    - Yard Cleanup
    - Hedge/Shrub Trimming
    - Leaf Removal
]

## Features
- Single Page Layout
- CSS Flexbox For Responsivness
- JavaScript Quote/Price Calculator
